# knn-breast-cancer-classification
Breast cancer classification using k-Nearest Neighbors

# Classificação de Câncer de Mama com KNN

Projeto de machine learning para classificação de câncer de mama utilizando o algoritmo K-Nearest Neighbors (KNN) e o dataset Breast Cancer Wisconsin.

## Funcionalidades
- Pré-processamento dos dados
- Análise exploratória
- Normalização dos dados
- Treinamento do modelo KNN
- Avaliação de acurácia

## Tecnologias
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Como Executar

Clone o repositório:

```bash
git clone https://github.com/AneDesiree/knn-breast-cancer-classification.git
```

Instale as dependências:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```

Execute o Jupyter Notebook:

```bash
jupyter notebook
```

Abra o arquivo:

```bash
breast_cancer_knn.ipynb
```

## Objetivo
O objetivo deste projeto é classificar tumores como malignos ou benignos utilizando o algoritmo KNN e analisar o desempenho do modelo.
